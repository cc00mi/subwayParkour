# 🏃 地铁酷跑 - Subway Rush

一款纯前端**地铁跑酷风格**单机小游戏，使用 HTML + Canvas + JavaScript 实现，无任何外部依赖。

## 在线游玩

直接打开 [`subway-rush.html`](./subway-rush.html) 即可在浏览器中运行。

或访问 GitHub Pages（需在仓库 Settings 中开启 Pages）：
```
https://fulloon1020.github.io/subwayParkour/subway-rush.html
```

## 操作方式

| 按键 | 动作 |
|------|------|
| `←` / `→` | 切换跑道 |
| `↑` | 跳跃 |
| `↓` | 下蹲 |

**移动端**：触摸屏幕滑动操作。

## 游戏规则

- 在三条跑道上不断向前奔跑，躲避障碍物
- **可破坏障碍物**（木箱）：碰撞后破碎，损失一条生命
- **不可破坏障碍物**（墙壁/横梁）：碰撞损失一条生命
- **跳跃**可越过木箱和墙壁
- **下蹲**可躲避横梁
- 初始 **3 条生命**，归零则游戏结束
- 收集金币可获得额外分数

## 障碍物类型

- 🟫 **木箱**（可破坏）：跳跃躲避
- 🧱 **墙壁**（不可破坏）：切换跑道或跳跃躲避
- ⚠️ **横梁**（不可破坏）：下蹲或切换跑道躲避

## 特点

- 伪 3D 透视跑道渲染
- 卡通可爱画风，代码绘制所有画面
- 渐变色天空 + 动态云朵 + 城市剪影
- Web Audio API 音效
- 粒子特效（碰撞、破坏、收集）
- 键盘 + 触摸双控制
- 最高分本地持久化存储

## 技术栈

- 纯 HTML + Canvas + JavaScript
- Web Audio API（音效）
- 无框架、无图片、无外部依赖

## 开发

单文件项目，直接编辑 `subway-rush.html` 即可。

```bash
# 本地预览（需要 HTTP 服务器）
python -m http.server 8080
# 打开浏览器访问 http://localhost:8080
```
